# QA Practice Project

## Описание
Данный проект содержит практические работы по тестированию веб-приложения.

В рамках проекта были протестированы:
- Регистрация пользователя
- Авторизация
- Восстановление пароля
- API
- База данных (SQL)

---

## 🔧 Используемые инструменты
- Postman (API тестирование)
- SQL (проверка данных в БД)
- GitHub (хранение тестовой документации)

---

## 📌 Что сделано

### ✅ Тест-кейсы
- Позитивные и негативные сценарии
- Проверка валидации полей
- Граничные значения

### 🐞 Баг-репорты
- Некорректный email
- Слабый пароль
- Дублирование пользователей
- Ошибки авторизации

### 🌐 API тестирование
- POST (создание пользователя)
- GET (получение данных)
- Проверка статус-кодов
- Негативные сценарии

### 🗄 SQL
- SELECT запросы
- WHERE фильтрация
- LIKE поиск
- Проверка данных после API

## 📊 Примеры навыков

- Написание тест-кейсов
- Составление чек-листов
- Поиск и описание багов
- Работа с API
- Проверка данных в базе
